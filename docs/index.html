API Info

You can use our API to accept BTCz payments on your website and verify the transaction.


Creating a gateway


  • URL: 
    https://pay.btcz.app/api/request_payment/:amount/:currency/:message/:seller_address/:customer_email/:srvPingback/:cliPingbackSuccess/:cliPingbackError
  • GET Example: 
    https://pay.btcz.app/api/request_payment/1/BTCZ/Hello/t1KhD2unS3KUjcVsANs5ySfxTVE9KTv6K4t/email%40test.test/http%253A%252F%252Fmysite.com%253A2222%252Fapi%252Ftest/http%253A%252F%252Fwww.worldartstamps.com/http%253A%252F%252Fbtcz.app
  • Issue note: Due to a API router issue, the 3 pinback URL's have to be encoded 2 time : 
    encodeURIComponent(encodeURIComponent('https://google.com'))
  • Sample Response: 
    {
  "id":"c5e9631d-b107-4022-8de5-ae9f0efd03af",
  "address":"t1gwku8spbCFUodyJ26njknnDxeZGM8hVmm",
  "link":"bitcoinz:t1gwku8spbCFUodyJ26njknnDxeZGM8hVmm?amount=14.77818972&message=Hello",
  "qr":"http://localhost:2222/generate_qr/bitcoinz%3At1gwku8spbCFUodyJ26njknnDxeZGM8hVmm%3Famount%3D14.77818972%26message%3DHello",
  "qr_simple":"http://localhost:2222/generate_qr/t1gwku8spbCFUodyJ26njknnDxeZGM8hVmm"
}

Fetching Data from Existing Gateway

  • URL: 
    https://pay.btcz.app/api/check_payment/:_id
  • GET Example: 
    https://pay.btcz.app/api/check_payment/4d30ed16-d729-4ee5-aeaa-d6787e79b90c

Other Information

  • States explained (you will only receive a pingback with states 2-errURL and 5-successURL):
    • 0: Awaiting payor to continue
    • 1: Awaiting payment
    • 2: Gateway expired
    • 5: Payment & Pingback completed
  • Standard tx fee: 0.0001
  • TX confirmation count: 2
  • State 5 execute the :srvPingback URL (on server side)
